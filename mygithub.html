AJUDA

$ git --help

Configurar usuário:

$ git config --global user.name "Alexandre Hesse"

Configurar email:

$ git config --global user.email "ale.hesse@gmail.com"

Verifica as configurações de nome e email:

$ git config user.name

$ git config user.email

Status do Arquivo:

$ git status

Inicializar Git:

$ git init

Adicionar arquivo para o Stage (segundo estado, palco):

$ git add teste.txt

$ git add . (1 ou mais arquivos)

Retirar arquivo do Stage (palco):

$ git rm --cached seuarquivo

Realizar o Commit:

$ git commit -m "identifique o seu commit"

Visualizar os commits:

$ git log

$ git log --oneline

Criar seu repositório no GitHub:

Acesse: https://github.com/AleHesse

  • No Overview, canto superior esquerdo, clicar em + > New Repository
  • Determinar o nome (Repository name) e definir Public or Private > criar Repository

Equipamento Local

Usando GitBash ou cmd:

  • Navegar até o local do diretório: cd.. ou cd suapasta ou c:
  • Mostra o path: $ pwd
  • Limpar tela: $ crtl + L
  • Criar pasta: $ mkdir suapasta
  • Criar arquivo: $ touch teste.txt
  • Visualizar diretório: $ ls
  • Visualização completa: $ ls -l

Usando VSCode: New Directory

  • Criar um novo diretório local
  • Abrir o diretório com VSCode
  • Dentro do VSCode, abrir o terminal
    echo "# somente_um_teste *opcional..." >> README.md  // opcional
    git init
    git add README.md                                  // opcional
    git commit -m "first commit"                       // first time only
    git branch -M main                                 // first time only
    git remote add origin https://github.com/AleHesse/teste.git   // first time only
    git push -u origin main                            // first time only

Usando VSCode: Diretório existente

  • Abrir o diretório com VSCode
  • Dentro do VSCode, abrir o terminal
  • No GitHub, criar um novo repositório
  • Obter o HTTPS do repositório: https://github.com/AleHesse/somente_um_teste.git
    …or push an existing repository from the command line
    git remote add origin https://github.com/AleHesse/teste.git
    git branch -M main
    git push -u origin main

Realizar o upload para GitHub:

$ git push

Após o primeiro push, apenas use:

$ git push (Upload)

$ git pull (Download)

Obs: Para sair do pager less (END) e voltar ao terminal, você pode pressionar a tecla q.

Visualizar o conteúdo de um arquivo:

$ cat seuarquivo.extensão

Excluir arquivo:

$ git rm nomedoarquivo.extensao

$ git commit - "Arquivo Excluído"

Recuperar arquivo:

$ git log --diff-filter=D --summary

$ git checkout 7b10~1 arq2.txt (primeiros dígitos do commit + ~1 nomearquivo)

Visualizar alterações:

$ git diff

Reverter commit:

$ git revert xxxxx (início do log do commit)

Uma janela será aberta para exibir a reversão... apenas feche.

Deletar projeto:

  • $ cd .. para sair da pasta do projeto
  • $ pwd (verificar o diretório)
  • $ rm -rf pastaDoProjeto
