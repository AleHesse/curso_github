Configurar usuário:
$ git config --global user.name "Alexandre Hesse"

Configurar email:
$ git config --global user.email "ale.hesse@gmail.com"

Localização:
$ pwd

Limpar tela:
$ crtl + L

Criar pasta:
$ mkdir suapasta

Entrar pasta:
$ cd suapasta

Visualizar diretório:
$ ls

Inicializar Git:
$ git init

Criar arquivo:
$ touch teste.txt

Visualização completa:
$ ls -l


Status do Arquivo:
$ git status
**Resultado**
On branch master // mostrar em qual ramo está

No commits yet	// nenhuma alteração ainda 

Untracked files:
  (use "git add <file>..." to include in what will be committed)
        teste.txt	//encontrado arquivo não rastreável

nothing added to commit but untracked files present (use "git add" to track) //nada foi adicionado ao commit(repositório) mas existe um arquivo não restreável precente
**


Adicionar arquivo para o Stage(segundo estado, palco)
$ git add teste.txt

$ git status  // para verificar

**Resultado**
On branch master
No commits yet
Changes to be committed: // agora existe alterações para serem commitadas
  (use "git rm --cached <file>..." to unstage)
        new file:   teste.txt // arquivo agora reconhecido
**


Retirar arquivo do Stage (palco)
$ git rm --cached seuarqivo


Realizar o Commit:
$ git commit -m "Primeiro Commit"

$ git status  //Verificar...
**Resultado**
On branch master
nothing to commit, working tree clean (nada para commitar... área de trabalho limpa)


Realizar o log:
$ git log  // dados de que realizou o commit

**Resultado**
